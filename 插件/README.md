# Smart Copy 安装包

本目录包含 Smart Copy 插件的预打包版本。

## 📦 包含的文件

- `smart-copy.zip`: **IntelliJ IDEA** 插件安装包。
- `smart-copy-1.0.0.vsix`: **VS Code** 扩展安装包。

## 🚀 安装指南

### IntelliJ IDEA
1. 打开 IDEA，进入 `Settings` -> `Plugins`。
2. 点击齿轮图标，选择 `Install Plugin from Disk...`。
3. 选择此目录下的 `smart-copy.zip` 并重启。

### VS Code
1. 打开 VS Code，进入扩展面板 (`Ctrl+Shift+X`)。
2. 点击右上角 `...` 菜单，选择 `Install from VSIX...`。
3. 选择此目录下的 `smart-copy-1.0.0.vsix`。
